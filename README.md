### 구현 할 기능들
1. 파도 차트 데이터로 표 만들기
2. 과거 파도 차트 저장 및 뷰 구현
3. 알림 기능
4. 카풀 기능 만들어보기
5. 이전 프로젝트 데이터 옮기기
